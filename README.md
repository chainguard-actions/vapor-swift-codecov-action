# vapor/swift-codecov-action

A GitHub Action which performs Codecov.io uploads with additional support for Swift projects.

Hardened by [Chainguard](https://www.chainguard.dev) from the upstream action at [https://github.com/vapor/swift-codecov-action](https://github.com/vapor/swift-codecov-action).
